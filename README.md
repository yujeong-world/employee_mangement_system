# 직원 관리 게시판 구현

직원 정보를 관리할 수 있는 게시판 프로젝트입니다.

## 기능 명세
1. 직원 목록 조회
   - 직원 목록 출력
   - 페이징 : 사용자는 한 페이지에서 볼 직원 정보의 수를 지정할 수 있다.(5, 10, 15, 20, 50개씩 보기)
2. 직원 등록
   - 등록시 사용자로부터 입력 받을 정보 : 직원번호, 직원명, 직급, 전화번호, 이메일
   - 직원번호는 유일한 값이어야 한다.
   - 직원명, 전화번호, 이메일은 정규표현식에 일치해야지 정상 등록 처리된다.
3. 직원 수정
   - 직원 등록시 입력 받았던 모든 필드에 대해 수정 가능
